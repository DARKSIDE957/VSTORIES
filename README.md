# VStories — Web Code

This repository contains the static website code for VStories.

Live website
- https://DARKSIDE957.github.io/VSTORIES/

What’s here
- index.html, stories.html, about.html
- stories/ (individual story pages)
- assets/css/styles.css
- assets/js/main.js, assets/js/data.js
- .nojekyll (ensures GitHub Pages serves files as-is)

How to update
1. Edit files in this folder.
2. Commit and push to the main branch.
3. GitHub Pages serves the latest version at the link above.
